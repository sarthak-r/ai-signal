GPT-4.5 and o3 retired from ChatGPT — but the API is unaffectedOpenAI · Jun 26–27

GPT-4.5 was removed from the ChatGPT model picker (and Custom GPTs) on 26 Jun after its 30-day sunset; o3 follows on 26 Aug after a 90-day sunset. Both changes are ChatGPT-product only — there are no changes to the API. If you build on the API, this weekend's ChatGPT retirements don't touch your integrations — no migration needed. The date that matters is 11 Dec: pin to a snapshot, not a floating alias, and schedule the move to a 5.5-class model before then so a silent cutover doesn't surprise you.
